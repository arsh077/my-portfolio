## üèÉ‚Äç‚ôÇÔ∏è Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd portfolio-website
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## üé® Customization

### Colors
The primary accent color is defined as `#C3E41D` (lime green). You can change this in the Tailwind config or directly in the components.

### Content
Update the content in `App.tsx`:
- Personal information in the hero and about sections
- Project details in the portfolio section
- Service descriptions
- Contact information

### Images
Your professional photos are stored in the `public` folder:
- `profile-red-bg.jpg` - Hero section image (stylish red background portrait)
- `profile-striped-shirt.jpg` - About section image (professional business look)
- Additional placeholder files for future use

**To add your actual photos**: Replace the placeholder files in the `public` folder with your actual images using the same filenames.
